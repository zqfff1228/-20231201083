## 数据模型关系思维导图

```
数据模型关系
├── 核心模型
│   ├── User (Django内置)
│   │   ├── 用户名
│   │   ├── 密码
│   │   ├── 邮箱
│   │   └── 活跃状态
│   ├── UserProfile (扩展信息)
│   │   ├── 头像
│   │   ├── 个人简介
│   │   ├── 所在地
│   │   ├── 发帖数
│   │   └── 评论数
│   ├── Category (分类)
│   │   ├── 分类名称
│   │   ├── 分类描述
│   │   └── 创建时间
│   ├── Post (帖子)
│   │   ├── 标题
│   │   ├── 内容
│   │   ├── 作者 (FK → User)
│   │   ├── 分类 (FK → Category)
│   │   ├── 标签 (JSON)
│   │   ├── 创建/更新时间
│   │   ├── 统计数据
│   │   └── 状态标记
│   └── Comment (评论)
│       ├── 内容
│       ├── 作者 (FK → User)
│       ├── 所属帖子 (FK → Post)
│       ├── 父评论 (自关联)
│       ├── 创建时间
│       └── 点赞数
├── 互动模型
│   ├── Like (点赞)
│   │   ├── 用户 (FK → User)
│   │   ├── 帖子 (FK → Post, 可选)
│   │   ├── 评论 (FK → Comment, 可选)
│   │   └── 创建时间
│   └── Favorite (收藏)
│       ├── 用户 (FK → User)
│       ├── 帖子 (FK → Post)
│       └── 创建时间
└── 关系映射
    ├── 一对多关系
    │   ├── User → Post (一个用户多个帖子)
    │   ├── User → Comment (一个用户多个评论)
    │   ├── Category → Post (一个分类多个帖子)
    │   └── Post → Comment (一个帖子多个评论)
    ├── 一对一关系
    │   └── User → UserProfile (用户与扩展信息)
    └── 多对多关系
        ├── User ↔ Post (通过Like模型)
        ├── User ↔ Post (通过Favorite模型)
        └── User ↔ Comment (通过Like模型)
```
